# 🌐 IPv4 Master Hub

A modern, responsive web application designed for **IPv4 Networking** education and utility. This tool bridges the gap between theoretical networking concepts and practical calculations, making it easier for students and professionals to validate IP structures.

---

## 🚀 Key Features
* **Subnet Calculator:** Quickly determine Dotted-Decimal masks and total host counts using CIDR prefixes.
* **IP Syntax Validator:** A real-time validator to ensure IPv4 addresses follow the correct 32-bit format.
* **Binary Logic Solver:** Uses bitwise operations to find the **Network (First)** and **Broadcast (Last)** addresses.
* **Adaptive UI:** Developed with a custom **CSS Grid** architecture that ensures a seamless experience across Mobile, Tablet, and Desktop.

---

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Grid Logic)
* **Logic:** JavaScript (ES6+ Bitwise Operators)
* **Frameworks:** W3.CSS (For clean, minimalist styling)

---

## 💡 Why I Built This
As a student and junior developer, I wanted to simplify the way we visualize IP addresses. This project was a great way to practice **Bitwise Logic** in JavaScript and master **Responsive CSS Grids** without relying on heavy frameworks.
