Protocol Overview

Internet Protocol Version 4 (IPv4) is the dominant protocol of the Network Layer in the OSI model. Introduced in 1983, it uses a 32-bit logical addressing scheme.

"IP provides a best-effort delivery mechanism, handling routing efficiently."

Hierarchical Addressing

IPv4 address parts: Network ID, Subnet ID, and Host ID. This hierarchy allows routers to manage data flow effectively.
